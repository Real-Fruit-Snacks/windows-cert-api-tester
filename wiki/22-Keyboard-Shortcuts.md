# 22. Keyboard Shortcuts

Shortcuts in the desktop app.

| Shortcut | Action |
|---|---|
| `Ctrl+Enter` / `Enter` | Send the request (Enter works in the URL (Uniform Resource Locator) box) |
| `Esc` | Cancel an in-flight request |
| `Ctrl+L` | Focus the URL box |
| `Ctrl+S` | Save the response to a file |
| `Ctrl+H` | Toggle the sidebar (collections / history) |
| `Ctrl+T` | New request tab |
| `Ctrl+W` | Close the current tab |
| `F5` | Refresh the certificate list |
| `F1` | Open Help & Reference |

## Also worth knowing

- **Middle-click** a request tab to close it.
- The **theme toggle** (☀ / ☾) and **Help** (`?`) live in the title bar.
- **Copy as ▾** (in the response toolbar, next to Copy body) copies the request as cURL / PowerShell / Python / C#.
- The response **find** box searches the body — Enter jumps to the next match and wraps around.
