# Certificate API Tester — starter endpoint wordlist for `certapi fuzz` / the Discover window.
#
# A curated list of paths worth probing on an undocumented API or web service: operational
# endpoints, API roots and docs, authentication, common resources, and paths that are often
# left exposed by mistake. It is a fast first pass — bring your own larger list (-w <file>) for
# a thorough sweep.
#
# Format: one entry per line. A bare path (/users) is probed with the chosen method(s); a
# "METHOD path" line pins a verb (POST /login). Blank lines and lines starting with # are
# ignored. Identical entries are de-duplicated automatically.
#
# Use this only against systems you are authorized to test.
